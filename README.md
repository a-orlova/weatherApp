# Weather App

https://weather-app-kappa-two-20.vercel.app/

веб-приложение для прогноза погоды с поддержкой геолокации. показывает текущую погоду и прогноз на несколько дней, позволяет добавлять несколько городов и обновлять данные без перезагрузки страницы. чистый js, адаптивная верстка
